# ERSP
Research on how LLM and AI can be verified when making medical decisions in triage and first aid situations.

Alloy folder: home to hand-coded alloy code for different triage scenarios. 

* Aila's Alloy Codes are following the wilderness first aid procedures outlines in https://troopone.org/wp-content/uploads/2021/07/T2-Troop-1-Wilderness-Medicine-Training-Overview-v4.pdf
